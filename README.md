# ZeroLimits Docs

Documentation for ZeroLimits projects.

## Build instructions

1. Clone the repo.

   ```sh
   git clone https://github.com/noClaps/docs.git
   cd docs
   ```

2. Install dependencies and start the development server.

   ```sh
   bun install
   bun dev
   ```

3. Build the site.

   ```sh
   bun run build
   ```

   This will build the site and place the files in `.vitepress/dist/`.
